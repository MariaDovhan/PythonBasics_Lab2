# PythonBasics_Lab2
Finding prime numbers in a user-specified interval

### Basic and individual task
The program:
<li>takes two numbers as start and end of an interval</li>
<li>checks if the start of the inteval is less than its end</li>
<li>uses for-loop to define if a number of an interval is prime</li>
<li>print out a number if it is prime</li>

### Additional functionality
A custom function that check if an input is a natural number and allow 3 input attempts
